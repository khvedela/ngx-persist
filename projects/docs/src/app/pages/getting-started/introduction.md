**NgxPersist** is a type-safe, signal-based persistent state primitive for Angular.

Manage your application state with the power of Angular Signals, automatically synced with `localStorage`, `sessionStorage`, or any custom backend. `ngx-persist` provides a seamless developer experience with full TypeScript support, making state persistence trivial in modern Angular applications.
